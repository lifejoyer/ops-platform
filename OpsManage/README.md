## 表结构说明

### project_assets： 
 * 项目表
 * id  项目id号，整型, 自增
 * project_name  项目名称, 字符串类型
 * project_english_name：项目英文名称，字符串类型

### server_assets:
 * 服务器主机表
 * id  id号，整型, 自增
 * ip： 主机IP地址，字符串类型
 * hostname：主机名称, 字符串类型
 * username：用户名称, 字符串类型
 * passwd：用户密码，字符串类型
 * port: 端口号, 整型
